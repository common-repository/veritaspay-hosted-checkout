Take payments on your store using VeritasPay Hosted Checkout

== Description ==

Accept Visa, MasterCard, GCash, GrabPay, ecPay, Alipay, and UnionPay on your store via VeritasPay Hosted Payment for WooCommerce.

== Installation ==

This payment gateway requires WooCommerce 6.1.1 and above.

= Installation =

1. Log in to your WordPress dashboard.
2. Hover to the **Plugins** and click the **Add New** button from the left sidebar.
3. In the search box on the upper right corner, type ***VeritasPay Hosted Checkout for WooCommerce*** and wait for it to load.
4. Once you found the **VeritasPay Hosted Checkout for WooCommerce** plugin, click **Install Now**.
5. You can now proceed on the **WooCommerce Payment Gateway Settings**
